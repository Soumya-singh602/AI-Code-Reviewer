# 🤖 AI Code Reviewer

An AI-powered code review application built with **Django REST Framework** and **React**. It uses the **Groq API** to analyze source code and provide intelligent suggestions to improve code quality, readability, and coding best practices.

---

## ✨ Features

* AI-powered code review using Groq API
* Real-time code analysis
* Clean REST APIs with Django REST Framework
* React-based user interface
* Docker & Docker Compose support

---

## 🚀 Getting Started

```bash
git clone https://github.com/Soumya-singh602/AI-Code-Reviewer.git

cd AI-Code-Reviewer

docker compose up --build
```

Open the application after the containers start.
